Your task is to write such a run-length encoding. For a given string, return a list (or array) of pairs (or arrays) [ (i1, s1), (i2, s2), …, (in, sn) ], such that one can reconstruct the original string by replicating the character sx ix times and concatening all those strings. 


function runLengthEncoding(str){
    let arr = [];
    let counter = 1;
    for(let i = 0; i < str.length; i++){
        if(str[i] === str[i+1]){
            counter++;
        }
        arr.push([counter, str[i]]);
        counter = 1;
    }
    return arr
}

Project description

The name of the project is "Elf against the trolls". This is a game, the main idea of ​​which is that an animated main character, an elf, collects coins and tries not to get caught by his enemies. Enemies are of three types - a troll (the most dangerous, the most points are removed for him), a knight is also dangerous, but less dangerous than a troll and birds.

The main character has a certain set of bullets (10), at the end of which you need to wait 7 seconds for new bullets to appear. There is also a saving mushroom, which can be sowed without being afraid of enemies for 5 seconds. At the end of the game, the results are sent to the database. By clicking on the "results" button you can get the results of the previous players. The game is rendered using canvas, characters are sprites. Technologies: HTML, CSS, JS.
